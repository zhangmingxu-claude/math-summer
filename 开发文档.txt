一、项目概述
-----------
单文件 HTML5 + PWA 数学学习平台，面向幼小衔接（5-7岁）。
- 22天系统学习日 + 8个周末生活数学日 = 30天闯关
- 每天含：小老师讲堂 → 三关练习 → 趣味擂台 → 错题回顾
- 支持添加到平板桌面，离线可用（Service Worker 缓存）
- 内置数字键盘（双键盘体系，练习/擂台各自独立）
- 25分钟休息提醒 + 护眼模式 + 家长学习报告

二、文件结构
-----------
math-summer/
  index.html       ← 唯一核心文件（~1700行），包含全部HTML/CSS/JS
  manifest.json     ← PWA 应用清单
  sw.js             ← Service Worker（离线缓存策略）

三、拼音注音实现
---------------
嵌入式映射表方案，零外部依赖：

1. PINYIN_MAP 对象（<head> 内）：
   - 792个汉字 → 拼音映射，覆盖全站所有中文
   - 数据来源：mozillazg/pinyin-data 开源词典
   - 生成方式见第四章

2. processPinyin() 函数：
   - 遍历讲堂 slide 的文本节点
   - PINYIN_MAP[ch] 查表 → 汉字包裹 <ruby> 标签
   - 表情/数字/英文/标点原样保留
   - data-pinyin-done 缓存标记

3. 拼音智能模式：
   - 学习日自动开启，周末自动关闭
   - 家长面板可设置为「总是开/关/智能」

四、如何更新拼音映射表
---------------------
前提：Python 3 + 网络

1. 提取全站汉字：
   python3 -c "
   with open('index.html','r',encoding='utf-8') as f:
       content = f.read()
   chars = sorted(set(ch for ch in content if '一' <= ch <= '鿿'))
   with open('_chars.txt','w',encoding='utf-8') as f:
       f.write(''.join(chars))
   print(f'{len(chars)} unique chars')
   "

2. 下载拼音词典（仅首次）：
   python3 -c "
   import urllib.request
   urllib.request.urlretrieve(
       'https://cdn.jsdelivr.net/gh/mozillazg/pinyin-data@master/pinyin.txt',
       '_pinyin_data.txt'
   )
   "

3. 生成映射表：
   python3 -c "
   import re
   pinyin_map = {}
   with open('_pinyin_data.txt','r',encoding='utf-8') as f:
       for line in f:
           m = re.match(r'U\+[0-9A-Fa-f]+:\s+(\S+)\s+#\s*(.)', line.strip())
           if m:
               py = re.sub(r'[0-9]', '', m.group(1).split(',')[0])
               pinyin_map[m.group(2)] = py
   with open('_chars.txt','r',encoding='utf-8') as f:
       chars = f.read().strip()
   entries = [f'{ch}:{pinyin_map[ch]}' for ch in chars if ch in pinyin_map]
   with open('_new_map.txt','w',encoding='utf-8') as f:
       f.write('const PINYIN_MAP={' + ','.join(entries) + '};')
   print(f'{len(entries)} entries')
   "

4. 复制 _new_map.txt 替换 index.html 中 PINYIN_MAP 块

五、内置数字键盘
---------------
双键盘体系，练习区和擂台区各一个：
- numPadPractice（id="numPadPractice"）：在 practiceArea 内
- numPadArena（id="numPadArena"）：在 arenaCard 内

核心函数：
- showNumPad(target) — 切换显示 target 对应键盘
- hideNumPad() — 隐藏两个键盘
- numPadInput/Delete/Confirm — 输入/删除/确认（通过 _numPadTarget 区分）
- _padSuffix() — 返回当前目标键盘的元素ID后缀

特性：
- 最大2位输入限制
- 单键删除（⌫只删最后一位）
- 确认键 ✓ 跨2行突出显示
- 数字/删除/确认键颜色区分
- 答错自动清空输入

六、擂台系统
-----------
activateArena() — 激活擂台（按钮 onclick）
submitArena()  — 提交答案（键盘✓或选项按钮调用）

按钮状态：初始"⚡ 进入擂台"→激活后"GO!"(禁用)→结束恢复
